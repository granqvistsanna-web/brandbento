<task type="auto">
  <name>Task 2: Add smooth color transition to html element</name>
  <files>src/index.css</files>
  <action>
Add a smooth transition for color scheme changes to the html element in the @layer base section.

Find the @layer base block and add this rule inside it (after the body rule):

```css
html {
  color-scheme: light;
  transition: background-color 200ms ease, color 200ms ease;
}

html.dark {
  color-scheme: dark;
}
```

Also update the body rule to include the transition:

```css
body {
  font-family: var(--font-sans);
  background-color: var(--canvas-bg);
  color: var(--canvas-text);
  line-height: 1.4;
  letter-spacing: -0.006em;
  overflow: hidden;
  transition: background-color 200ms ease, color 200ms ease;
}
```

This ensures:
1. Native form elements respect the color scheme
2. Background and text color transitions smoothly
3. No jarring flash when toggling theme
  </action>
